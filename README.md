# Cupcats Visual

A clean, calm, and minimal static website template designed for service-based brands.

This project serves two purposes:

1. A real brand website
2. A reusable website template

---

## Tech Stack

Pure static stack.

- HTML
- CSS
- Vanilla JavaScript

No framework.
No build step.
No bundler.

---

## Project Structure

/assets
  /css
  /js

  header.html
  footer.html

/pages

  booking

---

## Design Philosophy

The UI follows a calm and minimal style:

- relaxed
- simple
- quiet
- spacious

The brand voice avoids aggressive sales language.

---

layout  
components

responsive
