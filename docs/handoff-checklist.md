# Client Handoff Checklist

Use this checklist before delivering the website to a client.

## Branding

- [ ] Replace logo
- [ ] Update brand name
- [ ] Replace hero images
- [ ] Update favicon

## Content

- [ ] Update service descriptions
- [ ] Update contact information
- [ ] Update business address
- [ ] Update opening hours
- [ ] Update FAQ content

## SEO

- [ ] Update page titles
- [ ] Update meta descriptions
- [ ] Update canonical URLs
- [ ] Update OG image
- [ ] Verify sitemap.xml
- [ ] Verify robots.txt

## Booking System

- [ ] Google Sheet created
- [ ] Apps Script deployed
- [ ] Endpoint URL updated
- [ ] Booking test submitted
- [ ] Leading zero in phone number verified

## Domain

- [ ] Domain connected
- [ ] HTTPS enabled
- [ ] DNS records verified

## Final Testing

- [ ] Desktop layout
- [ ] Mobile layout
- [ ] All links working
- [ ] Booking form working
- [ ] Images loading correctly
- [ ] robots.txt accessible
- [ ] sitemap.xml accessible
